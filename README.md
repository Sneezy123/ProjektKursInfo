**Bisherige Features:** 
* Laufen
* Crouchen, Sprinten
* Interagieren
* Gegner mit KI
* Tür mit Interaktion
* Auto mit Interaktion
* Fadenkreuz mit Farbänderung bei Interaktion
* Sandbox Szene
